Proyecto de Flask de 1ºASIR, Gonzalo Nazareno. 
Json cogido de https://ceweb.br/guias/dados-abertos/es/capitulo-36/ y editado.
El JSON original estaba altamente anidado y duplicaba información.
La nueva versión aplica normalización, reduce la jerarquía, separa entidades y usa identificadores para relacionar datos, haciéndolo más escalable y reutilizable.
